# flutterfirebaseweblogin

A new Flutter project.

## Getting Started

### Set up

1. Add google-services.json file in build.gradle under android/app.
2. Go to ` firebase_options.dart` file and fill out the following files.

   - appKey
   - appKey
   - appId
   - messagingSenderId
   - projectId
   - storageBucket
   - measurementId
   - authDomain
   - iosClientId
   - iosBundleId
